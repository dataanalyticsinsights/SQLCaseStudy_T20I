# SQLCaseStudy_T20I

# T20I Cricket Match SQL Analysis Project

# Problem Statement

This project analyzes T20I cricket match results from 2024 using SQL. The goal was to understand team performance, winning patterns, match frequency, head-to-head records, and ground-wise success.

# Objective

The objective of this project was to use SQL to answer cricket analytics questions such as:

-Which team won the most matches?

-How teams rank by total wins

-Which teams perform best while chasing

-Which team has the highest average winning margin

-Monthly match distribution

-Head-to-head records between selected teams

-Win percentage for each team

-Most successful team at each ground

# Dataset

The dataset contains T20I match information with the following columns:

Team1,
Team2,
Winner,
Margin,
MatchDate,
Ground,
SQL Concepts Used,
SELECT statements,
WHERE filtering,
GROUP BY,
ORDER BY,
COUNT(), AVG(),
RANK() and DENSE_RANK(),
CASE statements,
CTEs,
Subqueries,
UNION ALL,
String functions: SUBSTRING(), LOCATE(),
Date conversion using STR_TO_DATE(),
Variables using SET

# Analysis Performed
Identified matches between two selected teams.
Found the team with the highest number of wins in 2024.
Ranked teams based on total wins.
Calculated the highest average winning margin in runs.
Listed matches where the winning margin was above the overall average.
Found the team with the most wins while chasing.
Created head-to-head records between selected teams.
Found the month with the highest number of T20I matches.
Calculated matches played and win percentage for each team.
Identified the most successful team at each ground.


# Key Learnings

Through this project, I learned how to clean and analyze cricket match data using SQL. I practiced ranking teams, converting text dates into date format, extracting numbers from text fields, using CTEs for multi-step queries, and calculating win percentages. This project helped me understand how SQL can be used for sports analytics and performance comparison.
